Editor VBS


1.O site é como se fosse um bloco de notas, mas o objetivo dele é programar usando a linguagem VBS que significa Visual Basic Studio Editon, a linguagem é um pouco complicada, mas tem como aprender (obviamente) para usar o site apenas escrever o código e quando terminar de escrever seu código, apenas baixar seu código, que tem um botão azul escrito baixar, quando for baixar, irá aparecer uma telinha em cima dizendo para você colocar o nome do arquivo e com .vbs no final.


2.Existe botões que podem lhe ajudar na codificação, os botões ajudam gerando códigos que podem ajudar tanto as pessoas que sabem codificar tanto as outras pessoas que não sabem codificar.


3.O site NÃO tem vírus, o nosso objetivo é fazer que as pessoas possam codificar a linguagem VBS muito mais FÁCIL e prático, e o exato código que a pessoa escreveu, irá ser apenas transformado em arquivo com o .VBS no final.


4.Esse site tem quase zero erro (não se sabe se tem um erro, pois se tiver não teremos que mudar o texto) pois saiba que todos os erros reportados serão removidos o mais rápido possível
